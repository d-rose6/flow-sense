# flow-sense
Sistema de IoT para monitoramento de consumo hídrico residencial em tempo real, utilizando ESP32 e sensor de fluxo de água com integração via HTTP para aplicativo mobile e plataforma web.

O Flow Sense é um projeto de IoT (Internet das Coisas) desenvolvido para monitorar o consumo de água residencial em tempo real, promovendo o uso consciente e o planejamento financeiro.

📝 Sobre o Projeto
Este projeto foi desenvolvido como parte do meu curso de graduação. Ele consiste em um dispositivo físico instalado na saída da caixa d'água que se comunica com um ecossistema de software para fornecer dados precisos ao usuário.

Hardware: Utilizamos um microcontrolador ESP32 e um sensor de fluxo para medir a vazão da água.

Software: Desenvolvemos um aplicativo móvel para monitoramento instantâneo e uma plataforma web que recebe dados via HTTP para exibir gráficos de consumo e históricos detalhados.

🚀 Status do Projeto
Atualmente, o protótipo do hardware e o aplicativo mobile já estão funcionais, permitindo a leitura e visualização dos dados. A próxima etapa foca na consolidação do dashboard web.
